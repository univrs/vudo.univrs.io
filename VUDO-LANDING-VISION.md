## Key SVG/3D Concepts

### 1. The Mycelium Network (Background)

```
- Procedurally generated network graph
- Nodes pulse with activity
- Edges grow and retract organically
- Depth via parallax layers (3-5 layers)
- Glow effects on active connections
```

### 2. Veve Patterns (Accents)

```
- Geometric patterns inspired by Haitian Veve
- Draw-on animation (stroke-dasharray)
- Appear at interaction points
- Each section has signature Veve
- Gold/white on dark substrate
```

### 3. The Mitan (Central Element)

```
- 3D sphere or toroidal form
- Internal fractal structure visible
- Radiates connection lines
- Responds to scroll/mouse
- Central to hero and CTA sections
```

### 4. Spirit Particles (Floating Elements)

```
- Small luminous dots
- Drift slowly with Brownian motion
- Cluster near content
- React to cursor proximity
- Represent distributed Spirits
```

### 5. Data Flow Visualization

```
- Animated paths showing data moving
- DOL code → compilation → execution
- Type-safe transformations visible
- Color-coded by operation type
```
