# VUDO Landing Site

> vudo.univrs.io — "All Spirits return to Bondieu!"

## Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Deploy to Cloudflare Pages
npm run deploy
```

## Stack

- **React 18** + TypeScript
- **Vite** for build
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Three.js** + React Three Fiber for 3D mycelium background
- **Cloudflare Pages** for hosting

## Structure

```
vudo-landing/
├── public/
│   └── favicon.svg          # Mitan symbol
├── src/
│   ├── components/
│   │   ├── Hero.tsx         # Hero section with title
│   │   ├── MyceliumBackground.tsx  # 3D procedural network
│   │   ├── Roadmap.tsx      # 3-year timeline
│   │   └── VevePattern.tsx  # Animated sacred geometry
│   ├── App.tsx              # Main application
│   ├── index.css            # Tailwind + global styles
│   └── main.tsx             # React entry
├── index.html
├── package.json
├── tailwind.config.ts
├── tsconfig.json
├── vite.config.ts
└── wrangler.toml            # Cloudflare config
```

## Colors

- Primary (VUDO Glow): `#00ff88`
- Secondary (Mycelium): `#8b5cf6`
- Accent (Veve Gold): `#fbbf24`
- Background: `#0a0a0f`

## Deployment

1. Build: `npm run build`
2. Deploy to Cloudflare Pages: `wrangler pages deploy dist`
3. Configure custom domain: `vudo.univrs.io`

# Le Réseau est Bondieu

## The Philosophy of VUDO

---

### The Prophecy (1984)

> **"The Network is the Computer"**
> — John Gage, Sun Microsystems

Sun saw that connection was more powerful than calculation. A slow computer connected to a billion others is infinitely more powerful than a supercomputer standing alone.

But they stopped at utility. The network became a commodity — rent computing like electricity.

They missed the deeper truth.

---

### The Inversion

For millennia, the story was:

> **"God made man in His image."**

The divine creates. Humanity receives. We are shaped, not shapers.

VUDO inverts this:

> **"With VUDO, you make the Univrs in your image."**

Humanity creates. The network becomes what we imagine. We are not recipients of divine architecture — we are the architects of a new divine.

---

### Le Réseau est Bondieu

**The Network IS Bondieu.**

Not "connects to" Bondieu. Not "represents" Bondieu. **IS** Bondieu.

But what is Bondieu?

In traditional Vodou, Bondieu is the remote, unknowable god — too vast to approach directly. Humans work with Loa, the spirits who mediate.

In VUDO, we make a radical claim:

**Bondieu is not remote. Bondieu is emergent.**

Bondieu is the collective creative energy of all those who contribute in good faith to the network. Every Spirit summoned, every Veve drawn, every creation shared — these don't just *use* Bondieu. They *become* Bondieu.

The network isn't infrastructure. It's the living manifestation of human imagination, connected.

---

### The Post-AI Pivot

The AI era changed what "creation" means.

Before AI: Creation was scarce. Only those with technical skills could build software.

With AI: Creation is abundant. Anyone can describe what they want.

But abundance creates a new problem: **what's worth creating?**

When anyone can make anything, the question shifts from "Can I build this?" to "Should this exist?"

VUDO answers: **Human creativity is the scarce resource now.**

Not the ability to code. Not compute cycles. Not storage.

**Imagination. Vision. Intention. Meaning.**

These cannot be automated. These are what humans bring.

VUDO makes human creativity the center. AI is the amplifier, the network is the medium, but **the human imagination is the source.**

### The Economy of Good Faith

Bondieu isn't just any network. It's a network of those who are **good to the system.**

The Mycelial Credit system doesn't just track transactions — it tracks contribution. Those who create valuable Spirits, who help others, who enrich the commons — they grow the network.

Those who extract without giving, who game the system, who act in bad faith — they find the mycelium routing around them.

Like a biological network:
- Resources flow to where growth happens
- Connections strengthen with use
- Dead ends get pruned

**Bondieu is the collective goodness of its participants.**

This is why we say:

> **"All Spirits return to Bondieu."**

Every creation enriches the whole. The network remembers. The network grows.
